Como Usar o Pad do Disroot

O Pad do Disroot é uma ferramenta de edição colaborativa de textos em tempo real. Abaixo, explicamos passo a passo como usá-lo:

1. Acesse a Ferramenta

A ferramenta já está incorporada nesta página. Basta rolar para cima e usar a interface do Pad do Disroot diretamente no site.

2. Comece a Escrever

No campo de texto grande, comece a digitar ou cole o conteúdo que deseja editar. Você pode usar a ferramenta para escrever documentos, fazer anotações ou colaborar em projetos.

3. Compartilhe o Link

Para colaborar com outras pessoas, compartilhe o link da página. Todas as alterações serão sincronizadas em tempo real para todos os participantes.

4. Use as Ferramentas de Formatação

O Pad do Disroot oferece opções básicas de formatação, como:

  • Negrito: Use **texto** para negrito.
  • Itálico: Use _texto_ para itálico.
  • Listas: Use - ou * para criar listas.
  • Títulos: Use # para títulos e ## para subtítulos.

5. Salve o Documento

O Pad do Disroot salva automaticamente o conteúdo enquanto você edita. Para garantir que o texto não seja perdido, você pode copiar e colar o conteúdo em um arquivo local.

6. Colaboração em Tempo Real

Todos os participantes que acessarem o link poderão editar o texto simultaneamente. As alterações são destacadas com cores diferentes para cada usuário.

7. Privacidade e Segurança

O Pad do Disroot é uma ferramenta segura porque:

  • Nenhum cadastro ou login é necessário.
  • O conteúdo é armazenado temporariamente no servidor.
  • Nenhum dado pessoal é coletado ou armazenado.

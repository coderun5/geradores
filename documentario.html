Bem-vindo Professor(a) ao Gerador de Prompt para Roteiros de Documentários Escolares

Esta ferramenta foi projetada para ajudá-lo a criar prompts personalizados que serão enviados por você para ferramentas de IA, como Chat GPT, Claude e DeepSeek, para gerar planos de aula completos sobre a criação de roteiros de documentários ou curtas-metragens escolares.

Como funciona este Gerador de Prompt:

Preencha os Campos do Formulário:

Cada campo no formulário foi criado para capturar informações essenciais que ajudarão a personalizar o prompt enviado às IAs.

Gere o Prompt:

Após preencher todos os campos, clique no botão "Gerar".

Envie o Prompt para as IAs:

Abaixo do prompt gerado, você encontrará links para as ferramentas de IA (Chat GPT, Claude e DeepSeek).

Preencha corretamente os campos abaixo para gerar um Prompt personalizado:

Prompt Gerado:

Gerar Plano de Aula com as AIs:

Clique em um dos links abaixo e escolha para qual IA você quer levar o seu prompt gerado:

Chat GPT

Uma ferramenta de IA da OpenAI, especializada em geração de texto e assistência em tarefas criativas.

Claude

Uma IA avançada da Anthropic, focada em segurança e assistência em tarefas complexas.

DeepSeek

Uma plataforma de IA para pesquisa e geração de conteúdo, ideal para exploração de dados e insights.

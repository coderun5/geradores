Bem-vindo ao Gerador de Prompt para Correção de Redação

Esta ferramenta foi projetada para ajudá-lo a criar prompts personalizados que serão enviados para ferramentas de IA, como ChatGPT, Claude e DeepSeek, para corrigir redações com base nas cinco competências da redação do Enem. O prompt gerado incluirá uma análise detalhada, apontando os erros, sugerindo correções e refazendo o texto da redação.

Como funciona:

1. Insira o texto da redação: Cole ou digite o texto que deseja corrigir.

2. Gere o prompt: Clique no botão "Gerar Prompt" para criar a instrução para a IA.

3. Envie para a IA: Use os links abaixo para enviar o prompt para as ferramentas de IA e obter a correção.

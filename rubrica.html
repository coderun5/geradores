Bem-vindo Professor(a) ao Gerador de Rubricas de Avaliação com IA

Esta ferramenta foi projetada para ajudá-lo a criar rubricas avaliativas personalizadas, que serão enviadas por você para ferramentas de IA, como Chat GPT, Claude e DeepSeek, para gerar rubricas no formato de tabela, adaptadas às suas necessidades.(Um prompt é uma instrução ou pergunta fornecida a uma ferramenta de IA para gerar uma resposta ou realizar uma tarefa.)

Como funciona este Gerador de Rubricas:

Preencha os Campos do Formulário:

Cada campo no formulário foi criado para capturar informações essenciais que ajudarão a personalizar o prompt enviado às IAs. Ao preencher os campos, você fornece detalhes que serão incorporados ao prompt, permitindo que as IAs gerem rubricas avaliativas no formato de tabela, mais precisas e relevantes.

Gere o Prompt:

Após preencher todos os campos, clique no botão "Gerar". O sistema criará um prompt baseado nas informações fornecidas, que será exibido na caixa de texto abaixo.

Envie o Prompt para as IAs:

Abaixo do prompt gerado, você encontrará links para as ferramentas de IA (Chat GPT, Claude e DeepSeek). Clique em um dos links para enviar o prompt diretamente para a IA correspondente e gerar a rubrica avaliativa.

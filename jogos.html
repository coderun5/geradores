Bem-vindo(a) ao Gerador de Jogos Educacionais com IA

Esta ferramenta foi projetada para ajudá-lo(a) a criar prompts eficazes para gerar jogos educacionais com IA. Preencha os campos abaixo para orientar a criação do jogo.

Como Usar o Código Gerado:

  1. Copie o Prompt: Clique no botão "Copiar Prompt" para copiar o texto gerado.
  2. Cole no DeepSeek ou Blackbox AI: Acesse o DeepSeek ou o Blackbox AI e cole o prompt na ferramenta.
  3. Gere o Jogo: Siga as instruções da IA para criar o jogo educacional.
  4. Salve o Código: Salve o código HTML gerado no Editor de Texto, e salve em um arquivo com a extensão .html (ex: jogo.html).
  5. Abra no Navegador: Clique duas vezes no arquivo salvo para abrir o jogo no navegador e jogar.
